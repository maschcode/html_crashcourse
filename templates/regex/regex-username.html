Areas ( Bereiche ) Quantoren
  • [ acf ] - One of the character a,c or f ( Minimum one )
  • [ ^acf ] - One character but not one of these a,c or f (negation)
  • [ 1-3a-cA-C ] - One of 1,2,3 or one of a,b,c,A,B,C ( Minimum one )
  • [ 1-3 ] [ a-c ] [ A-C ] - One of 1,2,3 then direct a,b or c and then direct one of A,B,C ( e.g 1bB , 3aC )
  • [ acf ]* - no one or many character of a,c or f ( e.g a, "empty" , accc, ffa , f , ... )
  • [ 1-3 ]+ - Minimum one of 1,2,3 or more of them
  • [ acf ]? - no one or only one character of a,c,f
  • [ 1-3 ]{ 3 } - exact three character of 1,2,3 not more not less ( e.g 111 , 123, 211, ... )
  • [ 1-3 ]{4, } - minimum 4 character of 1,2,3 or more ( e.g 1111, 12344, ...)
  • [ acf ]{ 2, 5 } - two till five character of a,c,f ( e.g aafff , ffaaa )
Group - Logic Character Classes ( short form )
  • Web( Engineering)? - allow is web or web engineering ( capturing group )
  • Web(Engineering|Technology) - allow more posibilties , web engineering and web technology is allow
  • \d - one number of 0 - 9 ( same [0-9] )
  • \D - negation of \d , allow all but not numbers of 0 - 9 ( same [^0-9])
  • \w - allow one numbers or alphabet characters or only underline character ( same [0-9a-zA-Z_] )
  • \W - negation of \w
  • \s - one unicode-space ( e.g space, linebreak, tab )
  • \S - negation of \s
